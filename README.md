# Laboratorio 2 - Diseño de Aplicaciones Móviles

## Descripción

Aplicación desarrollada en Flutter para el **Laboratorio 2 del curso de Diseño de Aplicaciones Móviles**.
El proyecto utiliza una **estructura de datos tipo lista** para gestionar lugares que el usuario desea visitar.

## Funcionalidades

* Agregar lugares
* Editar lugares
* Eliminar lugares
* Marcar lugares como visitados
* Buscar lugares

## Tecnologías

* Flutter
* Dart
